Пустой объект без полей занимает 16 байт.

Если объявить new User(1, "name"):
заголовок = 16 (как у пустого объекта)
поле age = 4 (на тип int)
поле name = 48:
    16 byte (загаловок объекта)
    4 byte (hash)
    4 byte (ссылка)
    16 byte (загаловок массива char[])
    8 byte (4 символа по 2 байта каждый)
итого: 68

Запуск программы с Person из предыдщуго задания:

опции vm:
-Xmx3m -Xms2m:
до 1000 объектов gc не вызвался
до 10000 объектов gc отрабатывает
до 50000 объектов gc отрабатывает, но выбрасывает:
Exception: java.lang.OutOfMemoryError thrown from the UncaughtExceptionHandler in thread "main"

опции vm:
-Xmx8m -Xms4m:
50000 объектов gc орабатывает без ошибок
100000 объектов gc вызывается, потом выбрасывает OutOfMemoryError
